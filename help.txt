AETHER SPATIAL ASSISTANT - HELP
================================

Quick Access
-----------
• Command Gesture: CALL (thumb + pinky) 
• Hotkey: CTRL+H
• System tray (if available)

Contents

KEYBOARD SHORTCUTS
------------------
CTRL+SPACE  Open/close UI
ESC         Close UI  
CTRL+1/2/3  Switch panels (System/Developer/Settings)
CTRL+H      Show this help
CTRL+TAB    Developer mode
CTRL+M      Normal mode
CTRL+P      Presentation mode

GESTURE CONTROL
---------------
OPEN PALM   Toggle UI (show/hide)
POINT       Move cursor to hand
PINCH       Click/select on target
FIST        Cancel current action
PEACE       Switch to next panel
THUMBS UP   Confirm action
CALL        Show this help ← (you are here!)
SWIPE L/R   Navigate panels
SWIPE U/D   Scroll content
GRAB        Drag window
RELEASE     Drop window
SCROLL_UP   Scroll up (two fingers up)
SCROLL_DOWN Scroll down (two fingers down)

MODE DISPLAY
------------
Top-right indicator shows current mode:
GREEN  NORMAL     - standard operation
YELLOW DEVELOPER - advanced tools
RED    PRESENTATION - kiosk mode
PURPLE AI        - AI features enabled

SYSTEM STATUS
-------------
Top-left shows CPU and RAM usage from the System panel.
Help overlay auto-hides after 10 seconds (toggle with 🔄 button).

WINDOW CONTROLS
---------------
• Click and drag header to move
• Minimize/Close buttons in top-right
• Resizable (drag bottom-right corner)

NAVIGATION TIPS
---------------
• Gestures work best within arm's reach
• Keep camera field of view clear
• Stay still 0.5 seconds between gestures
• Try: FIST to cancel, PEACE to switch panels
